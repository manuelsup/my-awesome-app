=========================================
How to Start the Website and Server
=========================================

1. STARTING THE FRONTEND (WEBSITE)
-----------------------------------------
Open a terminal or command prompt and navigate to the React app folder:
cd c:\Users\emi-a\Desktop\website\my-awesome-app

If this is your first time, or if you added new packages, run:
npm install

To start the Vite development server, run:
npm run dev


2. STARTING THE BACKEND (PYTHON SERVER)
-----------------------------------------
Open a separate, second terminal window and navigate to your Python backend folder.

Your React app is currently configured to connect to the backend at http://localhost:5000 (as seen in Login.jsx).

Run your Python server (the exact file name depends on what you named your Python script, usually app.py or main.py):
python app.py
